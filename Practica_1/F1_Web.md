
# ***Nombre del Sistema***
Sistema Web Informativo de Fórmula 1 (Web F1)

## ***Requerimientos del Sistema***
El sistema es una página web estática desarrollada en HTML y CSS que presenta información oficial sobre la Fórmula 1, incluyendo pilotos, resultados de carreras, próximos eventos y patrocinadores.

### ***1. Requerimientos Funcionales (RF)***
#### - RF1 – Búsqueda de Pilotos: 
    El sistema debe permitir al usuario buscar pilotos mediante un campo de texto con ejemplos sugeridos (Max, Oscar, Lewis).
#### - RF2 – Visualización de Pilotos:
    El sistema debe mostrar la información de cada piloto, incluyendo nombre completo, edad, país y monoplaza asignado para la temporada 2026.
#### - RF3 – Tabla de Resultados: 
    El sistema debe presentar una tabla con los resultados de las carreras de la temporada 2025, mostrando las posiciones obtenidas por Hamilton, Verstappen y Piastri en cada Gran Premio.
#### - RF4 – Próximos Eventos: 
    El sistema debe mostrar información de los próximos Grandes Premios, indicando fecha, hora local y ubicación del evento.
#### - RF5 – Navegación: 
    El sistema debe proveer una barra de navegación con enlaces al calendario oficial, equipos, patrocinadores y video de F1 2026.

### 2. Requerimientos No Funcionales (RNF)
#### - RNF1 – Usabilidad: 
    La interfaz debe ser intuitiva y visualmente clara, utilizando imágenes, iconos (Font Awesome) e identificación con banderas para facilitar la lectura del contenido.
#### - RNF2 – Compatibilidad: 
    El sistema debe ser compatible con distintos dispositivos y navegadores, haciendo uso de la etiqueta viewport para garantizar una visualización responsiva.
